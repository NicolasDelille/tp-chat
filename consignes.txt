CHAT AJAX

Infos sur la base de données COMMUNE à utiliser : 

	host : 10.0.0.8
	user : student
	pass : student
	name : chat

	Table `messages` :
		`id` int(11) NOT NULL,							clef primaire
		`message` varchar(255) NOT NULL,				contenu du message
		`username` varchar(50) NOT NULL,				pseudo de l'auteur du message
		`date_posted` datetime NOT NULL,				datetime de l'envoi du message
		`microtime_posted` decimal(14,4) NOT NULL		le temps unix en microsecondes d'envoi du message (à générer en PHP)

	**Vous n'avez les droits que pour faire des "INSERT" et des "SELECT"**

	Pour accéder au phpmyadmin de cette base : 
		http://10.0.0.8/phpmyadmin


Structure du chat : 

	Page d'accueil - index.php : 
		Formulaire de login avec simplement un champs pour le pseudo. Pas de mot de passe. 
		Sur réception du formulaire, le pseudo est conservé en session, et l'utilisateur est redirigé vers le chat. 

	Page du chat - chat.php : 
		La messages s'affichent dans une boîte avec un scrollbar vertical.
		Sous cette boîte, un champs de texte permet de créer un nouveau message. Un bouton "OK" est présent à côté du champ pour soumettre le message. 

	Envoi de message - send.php : 
		Fichier permettant d'insérer un nouveau message en bdd

	Récupération des messages - get.php : 
		Fichier permettant la récupération des messages
